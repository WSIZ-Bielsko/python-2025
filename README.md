## poetry installation


- insalacja przez pipx (zakładamy że mamy działającą conda); tu tworzymy nowy env condy jako base interpreter, ale można użyć
jakiegoś istniejącego

```
conda create -n pp12 python=3.12
conda activate pp12
pip install pipx

pipx install poetry
pipx ensurepath
```
